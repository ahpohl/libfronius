Detailed Description

Configuration for a shared Modbus bus.

Holds parameters that belong to the physical transport and are shared by every device on the bus: the transport descriptor, the reconnect policy, and the debug flag. Per-device parameters belong in ModbusDeviceConfig.

On RTU one instance covers all slaves on the same serial port; on TCP each remote host gets its own instance. Pass to FroniusBus.
