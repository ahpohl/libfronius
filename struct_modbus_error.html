Detailed Description

Encapsulates a Modbus error with code, message, and severity.

Used as the error type of std::expected<T, ModbusError> throughout libfronius. Severity is derived from the error code by deduceSeverity().

◆ deduceSeverity()

static Severity ModbusError::deduceSeverity ( int  c)
inlinestaticprivate

Map an error code to a Severity.

EINTR is mapped to SHUTDOWN (used to unwind blocking calls during shutdown). A fixed list of well-known fatal errno/libmodbus codes is mapped to FATAL; everything else is treated as TRANSIENT.
