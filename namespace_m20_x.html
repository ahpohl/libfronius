Detailed Description

Contains integer + scale factor SunSpec Meter registers for Fronius inverters.

All measurements in this namespace are represented as 16-bit integers. Scale factors must be applied to convert raw register values into physical units. Register addresses follow the standard SunSpec integer + scale factor model layout.
