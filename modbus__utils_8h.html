Detailed Description

Utility functions for working with Modbus registers and data.

Provides helper functions to extract integer, floating-point, and string values from Modbus register arrays, handle byte and word swaps, and convert values to hex strings. Includes validation for printable ASCII strings.

These functions are commonly used with Fronius SunSpec Modbus maps.
