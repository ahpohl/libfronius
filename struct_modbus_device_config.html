Detailed Description

Configuration for a single Modbus slave device.

Holds the slave ID, the response timeout the bus thread applies for this device, and the per-device reconnect policy used by scheduleDeviceRetry. Multiple devices on the same RTU bus each have their own instance with distinct slaveId values but share a single ModbusBusConfig.

Pass to Meter, Inverter, etc. alongside a shared_ptr<FroniusBus>.
