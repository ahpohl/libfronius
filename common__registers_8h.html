Detailed Description

Defines SunSpec Common Model (C001) register mappings for Fronius devices.

This header provides symbolic register definitions for the SunSpec Modbus Common Model block (C001), which describes general device information such as manufacturer, model, firmware versions, and serial number.

These definitions are used for Modbus communication with Fronius inverters, meters, and other compatible devices.
