Detailed Description

Holds all configuration options for a Modbus client.

Contains TCP/RTU connection settings, slave ID, debug flag, and reconnect parameters. Call validate() to ensure configuration values are within valid ranges.

void ModbusConfig::validate ( ) const
inline

Validate configuration parameters.

Exceptions
std::invalid_argumentif any parameter is out of allowed range.

Checks:

  • slaveId must be in 1–247
  • baud rate must be positive
  • TCP port must be 1–65535
  • reconnectDelay and reconnectDelayMax must be positive
  • reconnectDelay must be smaller than reconnectDelayMax
