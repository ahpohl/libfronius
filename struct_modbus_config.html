Detailed Description

Complete configuration for one Modbus device.

Combines a type-discriminated transport (TCP or RTU) with the per-device parameters that are independent of the physical bus: slave ID, response timeout, reconnect policy, and debug flag.

On RTU, the per-device slave ID allows multiple ModbusConnectionConfig values to share a single ModbusRtuBus (same device path) while each addressing a different slave.
