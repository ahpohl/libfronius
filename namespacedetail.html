Pack an integral value into consecutive 16-bit Modbus registers in big-endian word order.

Splits value into 16-bit words, byte-swaps each word from native little-endian to big-endian, and writes them into dest in most-significant word-first order, matching the SunSpec / Modbus-over-TCP wire format.

This is an internal helper. Callers should use ModbusUtils::packToModbus instead.

Template Parameters
TIntegral type to encode (e.g. uint32_t, uint64_t). Must satisfy std::is_integral_v<T>.
Parameters
destPointer to the first of sizeof(T)/2 consecutive uint16_t registers to write into. Must not be null and must have room for sizeof(T)/2 words.
valueValue to encode.
